# StoryBook Storefront

An AI-powered children's book storefront where users can browse, create, and purchase personalized stories generated by Claude AI.

## Quick Start

```bash
# Install all dependencies (root, client, server, e2e)
npm run install:all
cd e2e && npm install && cd ..

# Create .env in the project root
echo "ANTHROPIC_API_KEY=your-key-here" > .env
echo "PORT=3001" >> .env

# Start both client and server
npm run dev
```

The app runs at **http://localhost:5173** with the API at **http://localhost:3001**.

## Architecture

```
storybook-storefront/
├── client/          React 19 + Vite 8 + Tailwind CSS 4 frontend
├── server/          Express + TypeScript backend with Claude AI integration
├── e2e/             Playwright end-to-end tests
└── .env             ANTHROPIC_API_KEY and PORT
```

**Client** — Single-page app with React Router v7. Pages: Home (browse/filter), BookDetail (read pages), CreateBook (3-step AI wizard), Cart, Checkout, OrderConfirmation. Dark mode via ThemeContext with localStorage persistence. Cart state managed through CartContext with session-based IDs.

**Server** — Express REST API with JSON file persistence (`data.json`). Seeds 6 books with 5 pages each on first run. Story generation calls Claude API (`claude-sonnet-4-6`) to produce 5-page children's stories with structured JSON output.

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/books` | List books (query: `theme`, `age_range`, `featured`) |
| GET | `/api/books/themes` | Get available themes |
| GET | `/api/books/:id` | Get book with pages |
| POST | `/api/generate` | Generate AI story (body: `theme`, `characterName`, `ageRange`) |
| GET | `/api/cart/:sessionId` | Get cart items |
| POST | `/api/cart/:sessionId/items` | Add to cart (body: `bookId`) |
| PUT | `/api/cart/:sessionId/items/:bookId` | Update quantity |
| DELETE | `/api/cart/:sessionId/items/:bookId` | Remove item |
| DELETE | `/api/cart/:sessionId` | Clear cart |
| POST | `/api/orders` | Place order (body: `sessionId`, `customerName`, `customerEmail`) |
| GET | `/api/orders/:id` | Get order details |
| GET | `/api/health` | Health check |

## Scripts

### Root
```bash
npm run dev              # Start client + server concurrently
npm run dev:client       # Client only
npm run dev:server       # Server only
```

### Client (`client/`)
```bash
npm run dev              # Vite dev server on :5173
npm run build            # Production build
npm test                 # Vitest single run
npm run test:watch       # Vitest watch mode
```

### Server (`server/`)
```bash
npm run dev              # tsx watch mode on :3001
npm start                # tsx production start
npm test                 # Vitest single run
npm run test:watch       # Vitest watch mode
```

### E2E (`e2e/`)
```bash
npm test                 # Playwright headless (starts servers automatically)
npm run test:headed      # Playwright with browser visible
npm run test:ui          # Playwright interactive UI mode
npm run report           # Open last HTML report
```
